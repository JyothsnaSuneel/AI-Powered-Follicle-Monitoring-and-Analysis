# AI-Powered Follicle Monitoring and Analysis

## Overview

AI-powered healthcare application for automated follicle detection, segmentation, measurement, and maturity analysis from ultrasound videos using YOLOv8 instance segmentation.

## Business Problem

Manual follicle measurement in IVF treatment is time-consuming, subjective, and prone to inter-observer variability. This project automates follicle monitoring to support clinicians in treatment planning.

## Technologies

* Python
* YOLOv8m-seg
* OpenCV
* Streamlit
* NumPy
* Pandas
* Roboflow
* Deep Learning

## Features

* Ultrasound video upload
* Automated follicle segmentation
* Pixel-to-mm conversion
* Follicle maturity classification
* CSV report generation
* Streamlit clinical dashboard

## Model Performance

* Precision: 89.8%
* Recall: 90.2%
* mAP@50: 95.1%
* mAP@50-95: 73.6%
